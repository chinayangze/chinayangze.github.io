Win10 Subsystem Linux : Ubuntu 的root密码

每次开机都有一个新的root密码。我们可以在终端输入命令 sudo passwd，

然后输入当前用户的密码，enter，终端会提示我们输入新的密码并确认，

此时的密码就是root新密码。修改成功后，输入命令 su root，再输入新的密码就ok了。
