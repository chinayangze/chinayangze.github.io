使用 Hexo Admin 插件

使用方法

  1. 在Hexo网站目录下，安装 Hexo Admin 插件

npm install --save hexo-admin

  1. 启动本地服务器并打开管理界面，即可使用

hexo server -d open http://localhost:4000/admin/
